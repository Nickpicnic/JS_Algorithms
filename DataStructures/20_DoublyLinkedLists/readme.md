# Doubly Linked Lists

* Same as simple linked list but each node has a pointer to the previous one.
    - *implementation in doubliLinkedList.js*

* **Big O of Doubly Linked Lists:**
    - Insertion - **O(1)**
    - Removal - **O(1)**
    - Searching - **O(N)**
    - Access - **O(N)**
    - *Technically searching is **O(N/2)**, but that's **still O(N)***
