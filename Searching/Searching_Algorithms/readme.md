# Searching Alogithms
* JS builtin search methods:
    - indexOf
    - includes
    - find
    - findIndex

* **Linear Search**
    *implementation in linearSearch.js*
    - Linear Search **BIG O**:
        - O(1) *Best*
        - O(n) *Average*
        - O(n) *Worst*

* **Binary Search**
    *implementation in binarySearch.js*
    - Works only on sorted arrays!
    - Binary Search **BIG O**:
        - O(log n) *Worst and Average Case*
        - O(1) *Best Case*
